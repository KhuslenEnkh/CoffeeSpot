# ☕ CoffeeSpot

## 📌 Overview
CoffeeSpot is a web-based coffee shop review platform that allows users to discover cafes, browse by categories, and share their experiences through ratings and reviews.

## 👥 Team Members
- Khuslen Enkh-Amgalan
- Carlos Urra
- Usukhbayar Enkhbayar

## 🚀 Features
- Explore coffee shops by category
- Monthly top rankings
- View detailed coffee shop pages
- Submit reviews and ratings
- User authentication (Sign up / Login)
- Save favourite coffee shops

## 🗂️ Pages
- Home Page
- Categories Page
- Coffee Shop Details
- Write Review Page
- User Profile
- Favourites Page

## 🎨 Design
Wireframes were designed using Figma to ensure a clean and user-friendly interface.

## 🛠️ Tech Stack
- Frontend: [React / Next.js / HTML CSS JS]
- Backend: [Node.js / Express]
- Database: [MySQL / MongoDB]

## 📈 Future Improvements
- Map integration (Google Maps)
- Search & advanced filters
- Image upload for reviews
- Social features (likes/comments)
